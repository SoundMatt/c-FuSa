# DO-178C in c-FuSa

DO-178C (RTCA/EUROCAE) is the software standard for airborne systems. c-FuSa supports DO-178C evidence production across several commands.

## Command mapping

| DO-178C Artefact | cfusa Command |
|---|---|
| Software Configuration Index | `cfusa sci` |
| Software Accomplishment Summary | `cfusa sas` |
| Problem Reports (§11.17) | `cfusa pr` |
| Test Evidence | `cfusa verify` |
| Coverage (statement/decision/MC/DC) | `cfusa coverage --lcov` |
| Annex A gap report | `cfusa do178 --dal a|b|c|d` |
| Tool qualification | `cfusa qualify` |
| PSAC template | `cfusa template psac` |

## DAL coverage

| DAL | cfusa `do178 --dal` | Key additional requirements |
|---|---|---|
| A | `a` | MC/DC coverage required |
| B | `b` | Decision coverage required |
| C | `c` | Statement coverage required |
| D | `d` | Basic objectives |

## Workflow

```bash
cfusa do178 --dal b --output DO178_GAP.md
cfusa sas --output SAS.md
cfusa sci --output SCI.md
cfusa pr --list
cfusa coverage --lcov coverage.info --threshold 100 --mcdc
cfusa qualify --output QUALIFY.json
```
